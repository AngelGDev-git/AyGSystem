# A&G System — Sitio Web (GitHub Pages)

Este sitio es **100% estático** (HTML/CSS/JS) y está listo para subir a GitHub y publicar en **GitHub Pages**.

## Publicar en GitHub Pages
1) Crea un repositorio (ej: `agsystem-site`).
2) Sube todo el contenido de esta carpeta.
3) GitHub → **Settings → Pages**.
4) Source: **Deploy from a branch**.
5) Branch: **main** y carpeta **/(root)**.
